The Media FieldType

Skip to end of metadata
  • Created and last modified by Petar Spanja on May 23, 2014
Go to start of metadata

This FieldType represents and handles media (audio/video) binary file.

Description

This FieldType represents and handles media (audio/video) binary file.

It is capable of handling following types of files:

  • Apple QuickTime
  • Adobe Flash
  • Microsoft Windows Media
  • Real Media
  • Silverlight
  • HTML5 Video
  • HTML5 Audio

Hash format

The hash format mostly matches the value object. It has the following keys:

  • id
  • path (for backwards compatibility)
  • fileName
  • fileSize
  • mimeType
  • uri
  • hasController
  • autoplay
  • loop
  • height
  • width
